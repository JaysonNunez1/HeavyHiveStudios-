# Heavy Hive Studios - Website PRD

## Original Problem Statement
Create a website for Heavy Hive Studios - a recording studio that has been operating for 6-7 months. Services include recording, mixing, mastering, beat production, private event space rental, photography/video production, and learning programs. Currently clients book via Peerspace but future goal is direct booking. Display the studio, services, and photos for clients to see.

## Target Users / Personas
1. **Musicians/Artists** - Looking for professional recording services
2. **Music Producers** - Seeking mixing, mastering, and beat production
3. **Event Planners** - Looking to rent creative space for private events
4. **Content Creators** - Need photography and video production services
5. **Aspiring Producers** - Interested in learning audio engineering skills

## Core Requirements (Static)
- Single-page scrolling website
- Black/gold brand aesthetic
- Services showcase
- Pricing display (Recording, Mix & Mastering, Beat Production)
- Studio gallery
- Contact information (email, Instagram)
- Book Now CTA linking to Peerspace
